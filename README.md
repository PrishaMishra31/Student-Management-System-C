# Student-Management-System-C
Menu-driven Student Management System in C using structures and file handling to manage student records.
## Features
- Add new student records
- Display all records
- Search student by roll number
- persistent data storage using file handling

## Technologies Used
- C Programming  
- File Handling (fread, fwrite)  
- Structures

## Concepts Used
- Structures
- File Handling (fread, fwrite)
- Menu-driven programming

## How to Run

1. Compile the program:
```
gcc student.c -o program
```

2. Run the program:
```
./program
```
